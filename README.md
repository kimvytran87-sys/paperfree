\# PaperFree Literature Pipeline



A lightweight academic literature pipeline for:



\- Searching papers

\- Generating `papers.csv`

\- Downloading open-access PDFs

\- Extracting abstracts

\- Building literature review tables



\## Features



1\. Search papers with keywords and time filters

2\. Generate `papers.csv`

3\. Automatically download open-access PDFs

4\. Extract abstracts

5\. Generate:

&#x20;  - `literature\_reading\_table.csv`

&#x20;  - `methods\_models\_summary.csv`



\## Run



Activate environment:



"# paperfree"
